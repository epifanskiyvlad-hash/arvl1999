# V2RayTun subscription через GitHub

## Что загрузить

Загрузи в GitHub один из файлов:

- `vless-subscription.txt` - обычный список ссылок, чаще всего подходит.
- `vless-subscription-base64.txt` - base64-версия подписки, если клиент требует encoded subscription.

## Рекомендуемый вариант

1. Создай приватный репозиторий на GitHub, например `v2ray-sub`.
2. Загрузи туда файл `vless-subscription.txt`.
3. Открой файл на GitHub.
4. Нажми `Raw`.
5. Скопируй ссылку.
6. В V2RayTun добавь подписку по URL.

Ссылка будет выглядеть примерно так:

```text
https://raw.githubusercontent.com/USERNAME/v2ray-sub/main/vless-subscription.txt
```

## Как менять конфиг на всех устройствах

1. Открой файл `vless-subscription.txt` в GitHub.
2. Нажми edit.
3. Замени строку `vless://...` на новый конфиг.
4. Сохрани commit.
5. На устройствах нажми обновить подписку в V2RayTun.

## Важно

Если репозиторий публичный, конфиг сможет открыть любой, у кого есть ссылка. Для приватного репозитория raw-ссылка может не работать в некоторых клиентах без авторизации. Если V2RayTun не умеет читать приватный GitHub raw URL, используй secret gist или неприметный публичный репозиторий, понимая риск доступа по ссылке.
